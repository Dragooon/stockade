Quick Start

From clone to running in under 5 minutes.

Prerequisites

  • Node.js 22+
  • pnpm
  • Docker (for sandboxed agents)
  • An Anthropic API key

Install

git clone https://github.com/Dragooon/stockade.git
cd stockade
pnpm install

Configure

cp config/config.example.yaml config/config.yaml
cp config/proxy.example.yaml config/proxy.yaml

Add your API key:

mkdir -p config/secrets
echo "sk-ant-..." > config/secrets/anthropic-api-key
The config/secrets/ directory is gitignored. Secrets never leave your machine.

Run

Simple mode (no containers)

Set sandboxed: false on your agents in config.yaml, then:

pnpm start:orchestrator

Agents run in-process. Fast to iterate, no Docker needed. Less isolation.

Full stack (containers + proxy)

# Terminal 1: Start the credential proxy
pnpm start:proxy

# Terminal 2: Start the orchestrator
pnpm start:orchestrator

The orchestrator auto-provisions containers on first message to a sandboxed agent. Docker must be running.

Add Discord

Edit config/config.yaml:

channels:
  discord:
    enabled: true
    token: ${DISCORD_TOKEN}
    bindings:
      - server: "YOUR_SERVER_ID"
        agent: main
        channels: "*"

Set DISCORD_TOKEN in .env or your environment. The agent receives all messages in bound channels and uses its system prompt to decide when to respond vs stay silent.

Verify

# Run all tests
pnpm test

# Start the orchestrator with terminal channel
pnpm start:orchestrator
# Type a message — the main agent responds
